# 🎓 Tofendo School - FinTech Learning Platform

Plateformes éducatives complète pour apprendre le développement FinTech en Guinée avec Tofendo.

## 📚 À propos

Tofendo School est une plateforme d'apprentissage complète qui vous enseigne comment construire des solutions FinTech modernes. Vous apprendrez en pratiquant sur des projets réels inspirés de la plateforme Tofendo.

## 🎯 Objectifs d'Apprentissage

✅ Maîtriser **React 18** et les hooks modernes  
✅ Construire des **interfaces admin professionnelles**  
✅ Implémenter **l'authentification et l'OTP**  
✅ Créer des **dashboards avec graphiques**  
✅ Gérer l'**état complexe** avec React  
✅ Optimiser les **performances** (memoization)  
✅ Construire des **UIs responsive**  
✅ Intégrer des **APIs externes**  

## 📦 Modules du Cours

### 1. **Module 1: Fondamentaux React**
- État et Props
- Hooks (useState, useEffect, useMemo)
- Gestion des formulaires

### 2. **Module 2: Admin Dashboard**
- Architecture d'une interface admin
- Navigation et layout
- Graphiques avec Recharts
- Tables et listes de données

### 3. **Module 3: Authentification**
- Login avec email/mot de passe
- Vérification OTP
- Gestion de session
- Sécurité des données

### 4. **Module 4: Design & UI/UX**
- Tailwind CSS
- Design responsive
- Accessibilité (ARIA)
- Thèmes et couleurs

### 5. **Module 5: FinTech Specifics**
- Gestion des transactions
- Taux de change en temps réel
- KYC/AML compliance
- Intégration bancaire

## ⚡ Démarrage Rapide

### 1. Cloner le repository
```bash
git clone https://github.com/mansamoussa01/Tofendo-school.git
cd Tofendo-school
```

### 2. Installer les dépendances du projet principal
```bash
cd projects/tofendo-admin
npm install
```

### 3. Lancer le développement
```bash
npm run dev
```

L'application démarre sur `http://localhost:3000`

## ❓ FAQ
